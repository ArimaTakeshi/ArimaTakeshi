●各フォルダの内容
　・Hackathons制作番号_学年_お題
　※学校のイベントである「ハッカソン」とは、4人チームで当日に出題されるお題に沿って
　　10時間以内にゲームを1本完成させるというイベントで、制作したゲームです。
　
　
　・tool制作番号_学年_内容
　※制作をより効率よく進めるために開発したツールです。
　
　
　・Work制作番号_学年_ゲームジャンル
　※学校の授業のなかで、定められた期間でゲームを制作したゲームが入っております。
　
　
　・総集編PV・ポートフォリオ
　※総集編PVは今まで制作したゲームを３分間の動画にまとめたものです。
　
　
　※ポートフォリオには、今までの学校生活・参加したイベント・ゲーム制作への思い・
　　学校ではどのような事をしているか詳しくわかります。
